## 🆕 v2.0 새로운 기능

webnovel-engine v3.0에서 검증된 시스템을 무협 엔진으로 이식했습니다.

### 1️⃣ 5축 자가 검수 시스템
회차 본문을 5개 축으로 자동 채점하고 약점을 짚어내는 검수 엔진. 무협 가중치 적용.

| 축 | 가중치 | 검사 항목 |
|---|------|----------|
| **MATERIAL_USAGE** | 30% | 무공·문파·기보 등 기획 재료가 본문에 활용됐는가 |
| **CHARACTER_CONSISTENCY** | 20% | 12종 인물 역할이 평면화 안 됐는가 |
| **CLIFFHANGER_STRENGTH** | 15% | 9유형 클리프행어 적용 |
| **MISE_EN_SCENE** | **20%** ⭐ | 무공 시전·전투 묘사 (무협 특수 가중) |
| **MARKET_VIABILITY** | 15% | 시장 트리거·명대사·분량 |

### 2️⃣ 4가지 검수 모드
- `auto_until_25`: 1~25화 자동 + 26화부터 작가 결정 (★ 기본값)
- `transition_only`: 마음 흐름 전환점 회차만 자동 검수
- `all_auto`: 모든 회차 자동
- `all_manual`: 작가가 버튼으로 실행할 때만

### 3️⃣ 마음 흐름 3단계 자동 매핑
무협 남성향 표준 3단계 (각성·동기 발현 → 관문 통과·세력 확장 → 목표 진화·재정의) 자동 적용. 50화 기준 [5, 10, 32, 35]가 전환점으로 자동 감지됩니다.

### 4️⃣ IdeaSeed JSON 입력 시스템
Claude Max 무협 스킬에서 변환한 IdeaSeed JSON을 STEP 1 → 4번째 탭에 업로드하면 컨셉 카드로 자동 변환됩니다. 미결정 사항은 작가가 직접 답변 후 반영.

### 5️⃣ 12종 무협 인물 역할 분류
사부 / 사형 / 사매 / 숙적 / 정파 원로 / 사파 조력자 / 은인 / 의형제 / 배신자 / 무림맹주 / 마교주 / 천외천 — 각 역할에 4~5개 subtype 포함.

### 6️⃣ 주인공 5종 + 24개 subtype 확장
기존 5종 유지 + 각각에 4~5개 세부 유형 추가. 예: 회귀자 → 단일회귀형/다중회귀형/조건부회귀형/흑화회귀형/타인회귀형.

### 7️⃣ 25화 누적 대시보드
1~25화 자동 검수 후 평균 점수, 축별 평균, 자주 누락되는 재료, 클리프행어 분포를 시각화. 26화 이후 모드 결정 가이드 자동 제공.

## 🎬 3단계 파이프라인 (v2.0 강화)

### STEP 1 · CONCEPT
**4가지 입력 방식**:
1. 📄 기획서 업로드 (DOCX/HWP/PDF/TXT/MD)
2. 💡 아이디어 직접 생성
3. ✏️ JSON 직접 입력
4. **🌱 IdeaSeed JSON** ⭐ NEW — Claude Max 스킬 출력 자동 변환

### STEP 2 · BUILD-UP
- **Core Arc** (50화 비트)
- **Extension Arc** (확장 시즌: bridge / jump 모드)
- **Plant & Payoff Map** (떡밥 배치·회수 설계)
- **Character Bible** (12종 역할 분류 자동 매핑)

### STEP 3 · WRITING
- 회차별 플롯 설계
- 본문 집필 (5,000~5,800자 표준)
- 19금 ↔ 15금 변환
- 품질 체크 (정합성·문법·서사·분량)
- **🔬 v2.0 자가 검수 (5축)** ⭐ NEW
  - 규칙 기반 검수 (즉시·무료)
  - LLM 깊이 검수 (Sonnet)
  - 자동 재집필 (REDO 등급 시)
- **📈 25화 누적 대시보드** ⭐ NEW
- 독자 시뮬레이터 (5개 페르소나)
- DOCX 다운로드
